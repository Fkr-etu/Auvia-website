# Brief identité visuelle — Auvia

## 1. Présentation du projet

**Nom de marque : Auvia**

Auvia est un compagnon professionnel intelligent basé sur l'IA.

Il aide les professionnels réglementés (initialement les professionnels de santé libéraux) à rester à jour face à l'évolution constante de leur métier :

* nouvelles recommandations ;
* changements réglementaires ;
* protocoles professionnels ;
* bonnes pratiques ;
* obligations documentaires.

Auvia surveille les sources pertinentes, analyse les changements et transforme une masse d'informations complexes en actions simples et compréhensibles.

---

# 2. Vision de marque

Auvia n'est pas un simple outil de veille.

C'est un **compagnon de confiance** qui accompagne le professionnel dans l'évolution permanente de son métier.

La sensation recherchée :

> "Je ne suis plus seul face à la complexité réglementaire. Auvia veille pour moi."

---

# 3. Positionnement émotionnel

La marque doit transmettre :

* confiance ;
* sérénité ;
* intelligence ;
* accompagnement ;
* anticipation ;
* expertise.

Elle doit éviter une impression :

* froide ;
* administrative ;
* institutionnelle ;
* anxiogène ;
* trop médicale.

Auvia doit ressembler davantage à un assistant personnel professionnel qu'à un logiciel de conformité.

---

# 4. Origine du nom

Auvia s'inspire de racines latines :

* "auxilium" : aide, soutien, assistance ;
* "via" : chemin, voie.

Le concept :

> Auvia accompagne le professionnel sur son chemin métier.

---

# 5. Direction artistique recherchée

Créer une identité visuelle :

* premium ;
* moderne ;
* rassurante ;
* minimaliste ;
* technologique mais humaine.

Références d'esprit :

* Apple (simplicité, confiance)
* Notion (clarté, organisation)
* Linear (élégance SaaS)
* Calm (sérénité)
* Arc Browser (modernité)

Ne pas créer une esthétique "logiciel médical traditionnel".

---

# 6. Logo

Créer un logo facilement identifiable.

Contraintes :

* fonctionner en icône d'application mobile ;
* fonctionner en favicon ;
* être reconnaissable en petit format ;
* être adapté à une marque internationale.

Explorer plusieurs directions :

## Direction A — Le compagnon

Symbole évoquant :

* présence ;
* accompagnement ;
* lien ;
* guide.

Éviter les symboles évidents type :

* robot ;
* cerveau ;
* stéthoscope.

---

## Direction B — Le chemin

Inspiré de "via".

Évoquer :

* progression ;
* orientation ;
* évolution ;
* trajectoire.

---

## Direction C — La lumière / le signal

Auvia transforme le bruit informationnel en signal clair.

Évoquer :

* clarté ;
* révélation ;
* intelligence.

---

# 7. Couleurs

Palette souhaitée :

## Couleurs principales

Créer une palette autour de :

* bleu profond (confiance, expertise) ;
* vert doux ou turquoise (santé, évolution) ;
* blanc / tons clairs (simplicité).

Éviter :

* rouge médical ;
* bleu hôpital classique ;
* couleurs trop agressives.

---

# 8. Typographie

Style :

* moderne ;
* lisible ;
* humain ;
* professionnel.

Préférer :

* sans-serif contemporaine ;
* formes légèrement arrondies ;
* impression de proximité.

Le nom doit pouvoir s'écrire :

Auvia

avec une identité forte même sans symbole.

---

# 9. Ton de communication

Auvia parle comme un assistant expert.

Exemples :

"Votre métier évolue. Auvia vous accompagne."

"Les changements importants arrivent jusqu'à vous."

"Moins de bruit. Plus de clarté."

"Votre pratique reste toujours à jour."

# 11. Critère de réussite

En voyant Auvia, un professionnel doit ressentir :

"Cette application m'aide à maîtriser mon métier sans ajouter de complexité."

La marque doit inspirer :

**la confiance d'un expert, la simplicité d'un assistant, et l'intelligence d'une IA.**
