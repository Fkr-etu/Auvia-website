# Auvia — Product Specification

Version : 0.2  
Statut : Product Definition  
Produit : SaaS compagnon IA de surveillance métier et réglementaire

---

# 1. Vision métier

## Le problème

Les professionnels exerçant des métiers réglementés doivent maintenir en permanence leurs pratiques à jour.

Dans des domaines comme la santé, les obligations évoluent constamment :

- nouvelles recommandations nationales ;
- recommandations régionales ;
- changements réglementaires ;
- évolution des bonnes pratiques ;
- nouvelles publications scientifiques ;
- modifications des règles de prise en charge ;
- évolution des protocoles professionnels.

Aujourd'hui, cette veille repose principalement sur l'humain.

Le professionnel doit :

- surveiller des dizaines de sources différentes ;
- lire des documents longs et complexes ;
- identifier ce qui concerne réellement son activité ;
- décider s'il doit modifier ses pratiques ;
- mettre à jour ses protocoles internes.

Cette charge est devenue trop importante.

---

# 2. Le problème spécifique des protocoles professionnels

Les protocoles sont au cœur de la qualité et de la conformité d'une activité.

Dans un cabinet ou une structure professionnelle, on retrouve souvent :

- protocoles d'hygiène ;
- procédures de prise en charge ;
- fiches opératoires ;
- procédures administratives ;
- documents qualité ;
- recommandations internes.

Ces documents évoluent rarement de manière proactive.

Le fonctionnement actuel est souvent :

1. Une nouvelle recommandation officielle est publiée.
2. Le professionnel en entend parler éventuellement.
3. Il doit vérifier si cela concerne son activité.
4. Il doit comparer avec ses propres documents.
5. Il doit modifier ses protocoles.
6. Il doit informer son équipe.

Cette chaîne est longue, chronophage et source de risques.

---

# 3. La mission d'Auvia

Auvia devient le compagnon intelligent chargé de maintenir la pratique professionnelle à jour.

Sa mission :

> Surveiller l'évolution du métier, détecter les changements importants et accompagner la mise à jour des protocoles professionnels.

Auvia agit comme une mémoire professionnelle augmentée.

---

# 4. Positionnement produit

Auvia n'est pas une bibliothèque documentaire.

Une bibliothèque répond à :

"Je cherche une information."

Auvia répond à :

"Dis-moi ce qui a changé et ce que je dois faire."

---

Auvia n'est pas une simple newsletter.

Une newsletter diffuse :

- beaucoup d'informations ;
- sans personnalisation ;
- sans analyse d'impact.

Auvia filtre :

- ce qui concerne le professionnel ;
- ce qui est important ;
- ce qui nécessite potentiellement une action.

---

Auvia n'est pas un outil de conformité automatique.

La décision finale reste humaine.

Auvia aide le professionnel à :

- identifier ;
- comprendre ;
- prioriser ;
- préparer une action.

---

# 5. Promesse utilisateur

La promesse centrale :

> "Auvia veille sur l'évolution de votre métier et vous aide à garder vos protocoles toujours alignés avec les recommandations actuelles."

---

Exemple utilisateur :

Un chirurgien-dentiste possède un protocole interne concernant la prise en charge des patients à risque infectieux.

Une nouvelle recommandation nationale est publiée.

Aujourd'hui :

- il peut passer plusieurs mois sans la connaître ;
- son protocole peut devenir obsolète.

Avec Auvia :

1. La recommandation est détectée.
2. L'IA comprend le sujet.
3. Elle identifie que l'utilisateur est concerné.
4. Elle compare avec son contexte métier.
5. Elle explique l'impact potentiel.
6. Elle propose une action.

---

# 6. Vision long terme : le référentiel métier vivant

L'ambition d'Auvia est de créer un référentiel métier vivant.

Contrairement aux bases documentaires classiques, Auvia comprend :

- les sources officielles ;
- leur évolution dans le temps ;
- les relations entre recommandations ;
- les impacts sur les pratiques ;
- les documents internes des utilisateurs.

Avec le temps, Auvia devient une mémoire professionnelle personnalisée.

---

# 7. Première verticale : santé libérale

Le secteur médical est un premier marché idéal car il combine :

- forte pression réglementaire ;
- responsabilité importante ;
- nombreuses recommandations ;
- manque de temps ;
- besoin de traçabilité.

Premières professions potentielles :

- chirurgiens-dentistes ;
- vétérinaires ;
- pharmaciens ;
- infirmiers libéraux ;
- kinésithérapeutes.

---

# 8. Extension future

Le moteur Auvia est conçu pour être verticalisé.

Chaque métier possède :

- ses sources ;
- son vocabulaire ;
- ses protocoles ;
- ses règles.

Le même moteur peut ensuite être adapté à :

- professions juridiques ;
- experts-comptables ;
- métiers financiers ;
- immobilier ;
- sécurité ;
- industrie réglementée.

---

# 9. Différenciation stratégique

La valeur d'Auvia repose sur quatre piliers :

## 1. Surveillance continue

Auvia surveille automatiquement les sources pertinentes.

## 2. Compréhension métier

L'IA comprend le contexte professionnel de l'utilisateur.

## 3. Analyse d'impact

Auvia ne transmet pas uniquement une information.

Il explique :

"Pourquoi cela vous concerne."

## 4. Passage à l'action

Auvia aide à maintenir les protocoles et pratiques à jour.

---

# 10. Vision finale

Dans un monde où l'information professionnelle augmente plus vite que la capacité humaine à la suivre, chaque professionnel aura besoin d'un compagnon intelligent.

Auvia devient ce compagnon.

Il connaît l'évolution du métier.

Il protège la qualité de la pratique.

Il réduit la charge mentale.

Il permet aux experts de rester experts.
